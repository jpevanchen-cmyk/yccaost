## 一、与「野草系统」的关系

| 项 | 说明 |
|:---|:---|
| 系统名称 | **野草系统**（对外统一名称；旧称「无为系统」已废止） |
| 命名缘由 | **草根生长，不依附大平台**；店铺如野草，在各自土地上扎根 |
| 标识角色 | 视觉化上述理念：**在坚硬环境中仍生长**，与系统「去中心化、店铺独立」气质一致 |

标识不是装饰图案，而是把「野草」二字从**名字**变成**可被看见的故事**。

## 二、设计思维（核心）

### 2.1 两个象征

| 元素 | 象征 | 在品牌中的含义 |
|:---|:---|:---|
| **岩石** | 力量 | 环境严酷、压力大、体量巨大；现实经营中的硬条件 |
| **野草** | 希望 | 不依赖宽阔土壤；在缝隙里仍要活、仍要长 |

二者并置，表达：**不是温室里的顺利生长，而是缝隙中的顽强**——与中小店铺、独立节点在平台时代的处境相呼应。

### 2.2 画面要讲的三件事

1. **巨岩的一角**——岩石大，画面只取局部，暗示外面还有更大的世界。  
2. **风化细缝与微量泥土**——草不是凭空出现；缝里的土是它能活的**唯一依靠**。  
3. **极小的草与略露的根系**——草弱但不放弃；根抓住缝壁，体现「努力长出来」而非「摆在花盆里」。

### 2.3 比例原则

- 草（含少量可见根）与可见石面的视觉比重，目标约 **1 : 9**。  
- 石头占绝大部分，突出**力量的压迫感**与**环境的艰难**。  
- 草必须**小**，但在小图标等场景仍须**看得清**——过小则失去「希望」的传达，需在「弱小」与「可辨认」之间平衡。

### 2.4 为何做「两张图」而非一张

| 类型 | 用途 | 风格 |
|:---|:---|:---|
| **主品牌图** | 主页、介绍、上线氛围 | **写实微距摄影**——讲故事、建情绪 |
| **图形标 / Favicon** | 浏览器标签、收藏夹、小尺寸入口 | **极简扁平抽象**——认符号、秒识别 |

同一品牌、两种载体：大图重**氛围与故事**，小图重**符号与识别**。关联靠**细缝、冷石、黄绿小草**等共同元素，而非把小照片直接缩小。

## 三、设计文稿（创意 Brief · 定稿）

### 3.1 主体与构图

- 画面主体为**一整块巨大岩石的局部切面**（非整山全景）。  
- 岩石表面具**冷、硬**的矿物质感（偏灰蓝、偏冷灰），体现力量感。  
- 一条因**风化**形成的**极细裂缝**（发丝级、略弯曲，非刀刻宽沟）。  
- 缝内**少量泥土**，且**几乎与岩石表面齐平**——不是深槽，强调「仅缝隙可供生存」。  
- 从缝中长出**一株很小的野草**（新芽/幼苗），**2～3 根细根**略作显露，贴缝壁或扎进薄土。  
- 构图：**正方形 1:1**；草宜位于画面中**偏下**、沿裂缝生长。  
- 背景：**干净、简洁**，适合作为品牌/LOGO 参考图（主图可用浅灰渐变或中性底；小图标用纯色浅底）。

### 3.2 禁止与避免

- 无文字、无人物、无建筑（图形标专版尤其不得带字，否则标签缩小时不可读）。  
- 避免「花盆感」「花园感」「整山广角」。  
- 避免裂缝过宽——宽缝会削弱「顽强挤出」的叙事。  
- 避免草过大，占据画面主导（除非专门做卡通另稿）。

### 3.3 情绪与气质

- **安静、肃穆、自然光**。  
- 不夸张、不卡通、不霓虹。  
- 观者感受：**沉 + 一点亮**——石压得住，草点得住。

## 五、视觉统一规范（初版）

上线与改版时请尽量保持下列一致，以免「大图与小标不像一家」：

| 要素 | 建议 |
|:---|:---|
| **石色** | 冷灰蓝、偏硬（主图与图标同一色系，图标可略深） |
| **草色** | 黄绿色；**图标可比主图略亮 10%～20%**（小图可读性） |
| **草形** | 一茎 **两片对称小叶**；避免一张三叶、一张单叶 |
| **裂缝** | 极细、略弯；图标用**白色或浅色负形**表现缝 |
| **泥土** | 主图：薄、近齐平；图标：可收成缝底一条深色或省略 |
| **背景** | 主图：干净浅灰；图标：浅米白或浅灰纯色 |
