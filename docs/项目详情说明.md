## 1. 架构摘要

| 项 | 现状 |
|----|------|
| 类型 | 响应式网页（Django） |
| 配置包 | `wuwei_system` |
| 业务包 | `waimai` |
| 数据库 | SQLite（`db.sqlite3`），可迁移 |
| 支付 | 模拟 + 微信 Native + 现金；**退款待上线前最后一批** |
| 日志 | 技术文件日志 + `OperationAuditLog` 操作审计（A.12） |
| 不打包 | 非桌面 Inno 安装包 |

更细目录见 `架构与目录.md`。

## 3. 想改 X，优先看哪

| 想改什么 | 优先位置 |
|----------|----------|
| 全站网址 | `wuwei_system/urls.py`、`waimai/views.py` |

| 用户角色、订单字段 | `waimai/models.py`、`dine_models.py` |
| 生态登录 / 工作台登录 | `ecosystem_auth.py`、`shop_work_auth.py`、`views.py`（CustomLoginView、shop_work） |
| 卖家后台某一分区 | `views.seller_panel_section` + 对应 `*_handlers.py` / 模板 `seller/` |
| 堂食营业、桌码、拼桌 | `dine_seller_handlers.py`、`operating_helpers.py`、`table_*.py`、手册 A.11 |
| 菜单清单 / 多价 | `product_seller_handlers.py`、`menu_helpers.py`、`product_labels.py` |
| 服务员现场 | `waiter_handlers.py`、`waiter_helpers.py`、工作台模板 |
| 后厨看板 | `kitchen_handlers.py`、`kitchen_helpers.py` |
| 派单 / 骑手 | `dispatch_helpers.py`、`workbench_handlers.py`、骑手面板模板 |
| 支付下单 / 回调 | `waimai/payments/` |
| 操作审计 / 技术日志 | `audit_helpers.py`、卖家 `audit` 分区、手册 A.12 |
| 桌贴 PDF | `table_sticker_pdf.py` |
| 店铺码生成 | `shop_code_helpers.py`、手册 A.13 |
| 关页 / 超时 | `session_guard.py`、`static/waimai/session_guard.js` |
| 文案与页面外观 | `templates/waimai/`、`static/waimai/` |

## 4. 改代码协作约定（摘要）

- 须开发者口令 **「执行」** 或 **「开始」** 才改代码 / 过程文档 / 提交。  
- 注释用中文；改动最小化。  
- 业务规则以手册为准；进度以 E 盘 `2-开发进度.md` 为准。  
- 发现值得记的错误：先问是否写入 E 盘《开发错误与解决方案》。
