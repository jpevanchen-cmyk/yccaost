## 1. 总体结构（白话 · 当前）

- **配置工程**：`wuwei_system/` — 网站总开关（设置、网址总表、启动入口）。  
- **业务应用**：`waimai/` — 用户、店铺、订单、支付、工作台、模板与静态资源（**管店 + 堂食 + 配送等尚未拆插件**）。  
- **文档**：`docs/` + 仓库根若干「野草系统-*.md」。  
- **本地数据**：`db.sqlite3`、`logs/`（不进 Git）。

浏览器访问 → 网址总表分发 → `waimai` 里的页面逻辑 → 数据库 / 支付接口（**尚未**经「统一存储层唯一出入口」）。

## 5. 两套登录（架构要点 · 当前）

```text
买家 / 店主 ──► /accounts/login/     （野草生态会话）
店主 / 员工 ──► /s/<店铺码>/work/   （店铺工作台独立会话）
```

子账号不得从生态门登录；工作台退出不影响生态登录（反之亦然，按现实现）。  
目标态联络员凭证通道见手册 **A.15.4**（**代码未实现**）。
