# 野草系统（YecaoST）

去中心化的本地餐饮 / 外卖网页系统。第一阶段以**单店**为主：**堂食优先、外卖为辅**；店铺自管菜品、订单、服务员 / 后厨 / 骑手工作台与支付配置。

仓库文件夹名：`YecaoST`。对外名称：**野草系统**。

**在线主页（正式试跑）**：**[https://yichbo.com/](https://yichbo.com/)**  
（服务器主页；旧地址 `/directory/` 会自动跳转到此处。）

---

## 快速开始（本机开发）

1. 安装 **Python 3.11**（或兼容版本）。  
2. 在项目目录创建并启用虚拟环境 `.venv`，安装依赖：见 `requirements.txt` 与 `docs/环境与依赖清单.md`。  
3. 启动（示例）：

```text
.\.venv\Scripts\python.exe manage.py runserver
```

手机同一 WiFi 测试可用：

```text
.\.venv\Scripts\python.exe manage.py runserver 0.0.0.0:8000
```

4. 本机浏览器打开：`http://127.0.0.1:8000/`（**仅本机调试**；对外访问请用上文 **yichbo.com**）。

更细的本地地址列表见仓库根目录备忘 `服务器启动命令以及网址等.txt`（该文件不进 Git）。
